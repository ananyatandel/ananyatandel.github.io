Challenge 5: Accessibility Statement

Accessibility concerns that Challenge 3 might have is mouse accessibility. Since the checkbox and the radio button is so small, it's difficult to be able to select the appropriate boxes correctly for people with visual impairments. In addition, navigation links are also difficult because screen readers cannot detect it unless it’s coded correctly. A screen reader doesn’t know how to skip the navigation link so people with visual impairments can have problems understanding the content on the HTML page. Also, if there is an image on the page with no alt text, it’s hard for people to know what the image is showing. This can easily be solved by including alt text in your HTML code so it doesn't disproportionately affect visually-impaired people. Additionally, forms may not have correct labels and instructions for the inputs. This can easily be fixed by having an accessible label for each input for the reader that is large enough to read. The buttons should also be large and have clear text with high text contrast so it's readable.
